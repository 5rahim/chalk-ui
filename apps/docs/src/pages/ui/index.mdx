# UI Library

## Prerequisites

Your application should be configured with Tailwind.

## Installation

```
npm install ui @rahimstack/typesafe-form class-variance-authority
```

### Peer dependencies

```
npm install -d @tailwindcss/typography @tailwindcss/forms @headlessui/tailwindcss
```

### UI Context

UI Context is useful for localization.

```tsx
'use client'

import { UIProvider } from 'ui'

interface ClientProvidersProps {
   children?: React.ReactNode
}

export const ClientProviders: React.FC<ClientProvidersProps> = (props) => {
   
   const { children, ...rest } = props
   
   return (
      <UIProvider config={{ locale: 'fr' }}>
          {children}
      </UIProvider>
   )
   
}
```
