# Introduction

Chalk UI provides a command-line interface (CLI) tool designed to facilitate the integration of fully-functional and completely customizable pre-made Tailwind components into your project.<br/>

## Advantages

With Chalk UI you have complete control over your code. You can customize and adapt the components to suit your specific requirements or just use them as is.

## Primitives

The components use a variety well-established, maintained libraries with a good community around them.

- [Zag-js](https://zagjs.com/overview/introduction) - Framework-agnostic UI component logic
- [Radix](https://www.radix-ui.com/docs/primitives/overview/introduction) - UI component library for design systems
- [Headless UI](https://headlessui.com/) - Unstyled, fully accessible UI primitives
- And more
