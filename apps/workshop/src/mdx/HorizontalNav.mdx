# HorizontalNav

Responsive navigation items with dropdown menus and support for keyboard navigation.

## Installation

In order to install the component, run the following command:

```shell
npx @rahimstack/chalk-ui add horizontal-nav
```

Knowing the dependencies can help you customize or extend the component with additional functionality.

### mobilePlacement

The dropdown menu will become a modal on small screens, you can choose the placement of that modal using the `mobilePlacement` prop.
