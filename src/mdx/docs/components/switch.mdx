### Uncontrolled

- Input type: `checkbox`, Ref prop: `inputRef`
- Value type: `"on" | "off"`
- If not required, `"off"` will be submitted if the switch is not checked.
- If required, `"on"` will need to be the value of the checkbox (The switch needs to be checked).
