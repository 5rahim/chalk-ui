## Prerequisites

- Next.js / React
- Typescript
- Tailwind

To ensure proper configuration with Tailwind, it is recommended to set up your application using tools such as `create-next-app` or other suitable templates.
Alternatively, you can manually install Tailwind by following the instructions provided in the [documentation](https://tailwindcss.com/docs/installation).
